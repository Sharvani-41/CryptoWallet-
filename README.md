# CryptoWallet

CryptoWallet is a Spring Boot based backend application that simulates a cryptocurrency wallet system. It allows users to manage wallets, perform deposits, withdrawals, and track transaction history.

## Technologies Used
- Java
- Spring Boot
- Maven
- MySQL
- REST APIs

## Features
- Create and manage wallets
- Deposit cryptocurrency
- Withdraw cryptocurrency
- Transaction history tracking
- Admin dashboard

## Project Structure
src/main/java/com.wallet.cryptowallet
- controllers – REST API endpoints
- config – security configuration
- main application class

## How to Run
1. Clone the repository
2. Configure MySQL database in `application.properties`
3. Run the Spring Boot application
4. Access APIs using Postman or browser
